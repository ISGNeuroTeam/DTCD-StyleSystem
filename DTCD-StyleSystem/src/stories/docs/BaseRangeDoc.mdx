# BaseRange

Элемент для выбора числового значения из заданного диапазона, представленный тэгом `<base-range>`.

## Свойства/Атрибуты

| Название | Описание                         | Значение |
|----------|----------------------------------|----------|
| min      | Минимальное значение диапазона.  | Число    |
| max      | Максимальное значение диапазона. | Число    |
| step     | Шаг изменения значения.          | Число    |
| value    | Значение элемента.               | Строка   |

## События

| Название | Описание                                                                         |
|----------|----------------------------------------------------------------------------------|
| input    | Срабатывает при изменении значения элемента путем перетаскивания ползунка мышью. |

## Примеры

По умолчанию, элемент выбора имеет следующие настройки, которые можно переопределить с помощью соответствующих атрибутов:

* минимальное значение (min) - 0;
* максимальное значение (max) - 100;
* текущее значение (value) - 0;
* шаг изменения значения (step) - 1.

Для отлавливания события input необходимо добавить соответствующий слушатель на элемент. В качестве примера приведен код для VueJS:

<Source
language='html'
format={true}
dark
code={`
<template>
  <base-range @input="inputHandler"></base-range>
</template>
`}
/>

  },
},
`}
