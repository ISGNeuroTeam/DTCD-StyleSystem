# BaseColorPicker

Элемент для выбора цвета, представленный тэгом `<base-color-picker>`. По умолчанию, значение компонента равняется `#252230`.

## Свойства/Атрибуты

| Название | Описание                     | Значение   |
|----------|------------------------------|------------|
| value    | Задает значение компонента.  | Строка     |
| disabled | Отключает компонент.         | Логическое |
| label    | Заголовок для Color Picker.  | Строка     |

## События

| Название | Описание                                                                     |
|----------|------------------------------------------------------------------------------|
| input    | Срабатывает при изменении значения путем клика по цвету в выпадающем списке. |

## Слоты

| Название   | Описание                                                                           |
|------------|------------------------------------------------------------------------------------|
| label      | Заголовок для Color Picker (работает аналогично свойству `label`).                 |

## Примеры

По умолчанию список доступных цветов скрыт. Для его открытия нужно нажать на основной элемент компонента. Повторное нажатие на элементе или по какому-либо из раскрывшегося списка закроет данный список:

<Canvas>
  <base-color-picker></base-color-picker>
  <base-color-picker value="red"></base-color-picker>
</Canvas>

<Source
language='html'

Для отлавливания события input необходимо добавить соответствующий слушатель на элемент. В качестве примера приведен код для VueJS:

<Source
language='html'
format={true}
dark
code={`
<template>
  <base-color-picker @input="inputHandler"></base-color-picker>
</template>
`}
/>

<Source
language='js'

methods: {
  inputHandler(e) {
    console.log(e.target.value);
  },
},
`}
/>
