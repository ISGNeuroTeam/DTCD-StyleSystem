# BaseIconButton

Кнопка с иконкой, представленная тэгом `<base-icon-button>`.

## Свойства

| Название | Описание                    | Значение                                                                                         |
|----------|-----------------------------|--------------------------------------------------------------------------------------------------|
| disabled | Отлкючает кнопку            | Логическое                                                                                       |
| theme    | Варианты отображения кнопки | Массив с названиями тем: ['theme_primary' \| 'theme_secondary' \| 'theme_green' \| 'theme_red']. |
| type     | Тип кнопки                  | 'submit' \| 'reset' \| 'button' \| 'menu'                                                        |
| size     | Размер кнопки               | 'big' \| 'small'                                                                                 |                                                                                                       |

## Атрибуты

| Название | Описание                    | Значение                                                                                          |
|----------|-----------------------------|---------------------------------------------------------------------------------------------------|
| theme    | Варианты отображения кнопки | Название тем через запятую: 'theme_primary' \| 'theme_secondary' \| 'theme_green' \| 'theme_red'. |

## Слоты

| Название   | Описание                                                                        |
|------------|---------------------------------------------------------------------------------|
| (default)  | Слот по умолчанию. Отвечает за внутреннее содержимое кнопки. Обычно это иконка. |
