# BaseRadioGroup

Контейнер для элементов `<base-radio>`, предназначенный для формирования группы радиокнопок. Представлен тэгом `<base-radio-group>`.

## Свойства

| Название | Описание                     | Значение |
|----------|------------------------------|----------|
| value    | Значение группы радиокнопок. | Строка.  |

## Слота

| Название  | Описание                                               |
|-----------|--------------------------------------------------------|
| (default) | Слот по умолчанию. В него нужно поместить радиокнопки. |

## События

| Название | Описание                                                          |
|----------|-------------------------------------------------------------------|
| input    | Событие, происходящее при выборе какой-либо радиокнопки в группе. |
