# BaseDropdown

Представляет собой выпадающий список, в которое можно поместить любое содержимое. Представлен тэгом `<base-dropdown>`.

## Свойства

| Название  | Описание                                                                 | Значение                             |
|-----------|--------------------------------------------------------------------------|--------------------------------------|
| theme     | Варианты отображения выпадающего списка.                                 | Массив с названиями тем.             |
| opened    | Открыт или раскрыт выпадающий список.                                    | Логическое                           |
| alignment | Расположение скрытого содержимого по горизонтали.                        | 'right' \| 'center'                  |
| placement | Расположение скрытого содержимого относительно родительского контейнера. | 'top' \| 'rightStart' \| 'leftStart' |

## Атрибуты

| Название | Описание                                     | Значение                                        |
|----------|----------------------------------------------|-------------------------------------------------|
| theme    | Варианты отображения выпадающего списка.     | Название тем через запятую.                     |

## Слоты

| Название     | Описание                                                                                                                             |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------|
| (default)    | Слот по умолчанию. Отвечает за скрытое содержимое.                                                                                   |
| toggle-btn   | Слот в элементе, на который нужно кликнуть, чтобы отобразилось скрытое содержимое. Сюда можно поместить текст, кнопки, иконки и т.п. |
| icon-arrow   | Слот, в который можно поместить иконку вместо стандартной стрелки.                                                                   |

## События

| Название | Описание                                                                                                                       |
|----------|--------------------------------------------------------------------------------------------------------------------------------|
| toggle   | Срабатывает после открытия или закрытия выпадающего содержимого. В объекте события 'Event' в поле 'detail' есть поле 'opened'. |
