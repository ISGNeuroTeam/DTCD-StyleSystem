# DataCAD Font Icons

Шрифт с иконками, как [FontAwesome](https://fontawesome.com/). Сгенерирован с помощью сервиса [IcoMoon](https://icomoon.io/).

## Примеры

Чтобы добавить иконку на страницу, нужно подключить данный шрифт. Затем добавить тег с классом `FontIcon` и класс с названием иконки:

<Canvas>
  <span className="FontIcon name_checkAll"></span>
  <span className="FontIcon name_flagFill"></span>
</Canvas>
<Source
language='html'
format={true}
dark
code={`
<span class="FontIcon name_checkAll"></span>
<span class="FontIcon name_flagFill"></span>
`}
/>

Размер иконок задается css-правилом `font-size` и по-умолчаниию наследуется от родительского тега. Чтобы изменить размер, нужно добавить класс-модификатор. Модификаторы соответствуют размерам текста из UI-kit. Так как иконки являются шрифтом, то они стилизуются, как текст.

## Название классов

Классы иконок сделаны по методологии [БЭМ в стиле "React"](https://ru.bem.info/methodology/naming-convention/#%D1%81%D1%82%D0%B8%D0%BB%D1%8C-react). Пример:

<Source
language='html'
format={true}
dark
code={`
<span class="FontIcon name_pieChart75 size_3xl"></span>
// FontIcon - название блока.
// name_pieChart75 - модификатор блока, где 'name' - название модификатора, а 'pieChart75' - его значение.
// size_3xl - модификатор блока, где 'size' - название модификатора, а '3xl' - его значение.
`}
/>

## Добавление/редактирование иконки

1. Импортировать проект в [https://icomoon.io/app/#/projects](https://icomoon.io/app/#/projects);
2. Импортировать .svg кнопкой "Import Icons" или перетащить в окно;
3. Редактировать каждую иконку:
    * Дать одно название пиктограммы иконке в Names (в формате camelCase);
    * Обесцветить - Color => Remove Color;
    * Задать Grid = 24;
    * Отцентрировать - Canvas/Alignment => Align to Center;
    * Если соотношение сторон 1:1 - Canvas/Alignment => Square Canvas;
    * Ужать чтобы она помещалась - Canvas/Alignment => Fit to Canvas;
4. Инструментом Select (в хедбаре) активировать иконку;
5. Нажать Generate Font в тулбаре снизу;
6. Также заместить папку /assets/fonts/ папкой /fonts/ из архива.
