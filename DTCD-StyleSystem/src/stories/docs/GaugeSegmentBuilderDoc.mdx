# GaugeSegmentBuilder

Особый компонент, предназначенный для формирования диапазонов для настройки визуализации Gauge, представленный тэгом `<gauge-segment-builder>`.

## Свойства

| Название | Описание                   | Значение                                  |
|----------|----------------------------|-------------------------------------------|
| value    | Текущее значение элемента. | Массив объектов (см. [примеры](#примеры)) |

## События

| Название | Описание                                                                            |
|----------|-------------------------------------------------------------------------------------|
| input    | Срабатывает при добавлении или удалении сегментов, а также изменении их диапазонов. |

## Примеры

По умолчанию компонент не имеет сегментов и его значение является пустым массивом `[]`. При нажатии на кнопку “Добавить диапазон”, в результирующий массив значения добавляется объект сегмента следующего вида:

<Source
language='js'
format={true}
dark
code={`
{ range: [0, 10], color: '#555' }
`}
/>

Ключи объекта сегмента:
* color - цвет сегмента;
* range - массив из двух элементов, которые обозначают начало и конец сегмента соответственно.

В случае, если значение начала диапазона сегмента больше его конца, появляется соответствующее предупреждение.

Для отлавливания события input необходимо добавить соответствующий слушатель на элемент. В качестве примера приведен код для VueJS:

<Source
language='html'
format={true}
dark
code={`
<template>
  <gauge-segment-builder
    @input="inputHandler"
  ></gauge-segment-builder>
</template>
`}
/>
<Source
language='js'
format={true}
dark
code={`
methods: {
  inputHandler(e) {
    console.log(e.target.value);
  },
},
`}
/>
