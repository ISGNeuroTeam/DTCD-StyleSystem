# BaseDateTimePicker

Компонент для установки даты и времени.

## Свойства

| Название    | Описание                                                                                                                               | Значение   |
|-------------|----------------------------------------------------------------------------------------------------------------------------------------|------------|
| visible     | Календарь открыт или скрыт.                                                                                                            | Логическое |
| value       | Получение и установка дат или даты через таймстампы.                                                                                   | Строка со таймстампами через точку с запятой или число |
| disabled    | Свойство поля ввода BaseInput.                                                                                                         | -          |
| required    | Свойство поля ввода BaseInput.                                                                                                         | -          |
| label       | Свойство поля ввода BaseInput.                                                                                                         | -          |
| size        | Свойство поля ввода BaseInput.                                                                                                         | -          |
| range       | Предоставляет возможность выбора диапазона дат. Атрибут `data-range`.                                                                  | Логическое |
| timepicker  | Включает возможность выбрать время. Атрибут `data-timepicker`.                                                                         | Логическое |
| timewindows | Отображать или нет блок кнопок с временными отрезками. Атрибут `data-timewindows`. Свойства `range` и `timepicker` должны быть `true`. | Логическое |

## События

| Название | Описание                           |
|----------|------------------------------------|
| input    | Событие подтверждения выбора даты. |
