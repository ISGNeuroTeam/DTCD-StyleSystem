# BaseSwitch

Элемент переключателя, позволяющий включить или отключить какую-либо опцию, подобно элементу `<base-checkbox>`. Представлен тэгом `<base-switch>`.

Элемент может быть активирован и деактивирован, что соответствует значению свойства `value` - `true` или `false`.

## Свойтва/Атрибуты

| Название | Описание                                                                     | Значение              |
|----------|------------------------------------------------------------------------------|-----------------------|
| value    | Значение переключателя ([документация](https://webref.ru/html/input/value)). | Строка, число         |
| checked  | Активирует / дезактивирует переключатель.                                    | Логическое            |
| disabled | Отлкючает переключатель.                                                     | Логическое            |
| type     | Тип поля ввода (`<input/>`).                                                 | `checkbox` \| `radio` |
| label    | Подпись переключателя.                                                       | Строка                |

## События

| Название | Описание                                                                   |
|----------|----------------------------------------------------------------------------|
| input    | Срабатывает при активации или деактивации переключателя при клике по нему. |
| change   | Срабатывает при изменении состоянии переключателя.                         |

## Слоты

| Имя слота  | Описание                                                |
|------------|---------------------------------------------------------|
| label      | Аналогично свойству `label`.                            |
