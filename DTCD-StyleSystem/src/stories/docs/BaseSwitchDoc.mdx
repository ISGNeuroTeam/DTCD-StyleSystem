# BaseSwitch

Элемент переключателя, позволяющий включить или отключить какую-либо опцию, подобно элементу `<base-checkbox>`. Представлен тэгом `<base-switch>`.

Элемент может быть активирован и деактивирован, что соответствует значению свойства `value` - `true` или `false`.

## Свойтва

| Название           | Описание                                  | Значение   |
|--------------------|-------------------------------------------|------------|
| value (deprecated) | Активирует / дезактивирует переключатель. | Логическое |
| checked            | Активирует / дезактивирует переключатель. | Логическое |
| disabled           | Отключает переключатель.                  | Логическое |

## События

| Название | Описание                                                                   |
|----------|----------------------------------------------------------------------------|
| input    | Срабатывает при активации или деактивации переключателя при клике по нему. |

## Примеры

Переключатель в активированном и активированном состоянии:

<base-switch></base-switch>
<base-switch checked></base-switch>

```
<base-switch></base-switch>
<base-switch checked></base-switch>
```

Заблокированные с помощью атрибута disabled переключатели в активированном и деактивированном состоянии:

<base-switch disabled></base-switch>
<base-switch disabled checked></base-switch>

```
<base-switch disabled></base-switch>
<base-switch disabled checked></base-switch>
```

Для отлавливания события input необходимо добавить соответствующий слушатель на элемент. В качестве примера приведен код для VueJS:

```
<template>
  <base-checkbox @input="inputHandler"></base-checkbox>
</template>
```
```
methods: {
  inputHandler(e) {
    console.log(e.target.value);
  },
},
```
