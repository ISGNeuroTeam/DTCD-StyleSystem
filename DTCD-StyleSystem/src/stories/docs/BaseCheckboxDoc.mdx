# BaseCheckbox

Элемент, позволяющий включить или отключить какую-либо опцию. Представлен тэгом `<base-checkbox>`. Чекбокс может быть активирован и деактивирован, что соответствует значению свойства `checked` - `true` или `false`.

## Свойства/Атрибуты

| Название | Описание                                                                 | Значение                    |
|----------|--------------------------------------------------------------------------|-----------------------------|
| checked  | Активирует / дезактивирует чекбокс.                                      | Логическое                  |
| value    | @deprecated. Тоже самое, что и `checked`.                                |                             |
| disabled | Отлкючает чекбокс.                                                       | Логическое                  |
| type     | Тип поля ввода (`<input/>`).                                             | `checkbox` \| `radio`       |
| label    | Подпись чекбокса.                                                        | Строка                      |
| placement | Расположение скрытого содержимого относительно родительского контейнера.| 'rightStart' \| 'leftStart' |

## События

| Название | Описание                                                              |
|----------|-----------------------------------------------------------------------|
| input    | Срабатывает при активации или деактивации чекбокса при клике по нему. |
| change   | Срабатывает при изменении состоянии чекбокса.                         |

## Слоты

| Название   | Описание                                                                     |
|------------|------------------------------------------------------------------------------|
| (default)  | Слот по умолчанию. Подпись чекбокса (работает аналогично свойству `label`).  |

  <base-checkbox label="Title"></base-checkbox>
</Canvas>
