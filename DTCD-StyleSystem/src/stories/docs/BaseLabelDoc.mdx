# BaseLabel

Цветной компонент-лейбл, представленный тэгом `<base-label>`.

## Атрибуты

| Название | Описание            | Значение                                       |
|----------|---------------------|------------------------------------------------|
| color    | Задает цвет лейбла. | 'accent' \| 'warning' \| 'danger' \| 'success' |

## Слоты

| Название  | Описание                                                     |
|-----------|--------------------------------------------------------------|
| (default) | Слот по умолчанию. Отвечает за внутреннее содержимое лейбла. |

## Пример

По умолчанию лейбл содержит текст “Label”, который может быть заменен с помощью слота:

<base-label></base-label>
<base-label>Custom Label</base-label>

```
<base-label></base-label>
<base-label>Custom Label</base-label>
```

Цвет лейбла по умолчанию - серый, но может быть изменен с помощью соответствующего атрибута color:

<base-label></base-label>
<base-label color="accent"></base-label>
<base-label color="warning"></base-label>
<base-label color="danger"></base-label>
<base-label color="success"></base-label>

```
<base-label></base-label>
<base-label color="accent"></base-label>
<base-label color="warning"></base-label>
<base-label color="danger"></base-label>
<base-label color="success"></base-label>
```
