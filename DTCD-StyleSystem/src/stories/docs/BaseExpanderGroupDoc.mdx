# BaseExpanderGroup

Компонент для объединения раскрывающийся списков (`<base-expander>`), представленный тегом `<base-expander-group>`. Объединять экспандеры нужно для того, чтобы из всей группы был раскрыт только одни экспандер, а остальные автоматически закрывались.

## Слоты

| Название | Описание                                                            |
|----------|---------------------------------------------------------------------|
| item     | Слоты, в которых содержатся раскрывающиеся списки (base-expander).  |
