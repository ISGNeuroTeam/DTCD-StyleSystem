# DataSourceSelect

Особый компонент, предназначенный для выбора источника данных для визуализаций. Компонент является полем ввода, реализован на основе выпадающего списка BaseSelect и представлен тэгом `<datasource-select>`.

Внутри компонента используется адаптер EventSystem, который используется для обновления актуального списка источников данных.

## Свойства

| Имя свойства | Описание                            | Значение                        |
|--------------|-------------------------------------|---------------------------------|
| value        | Текущее значение внутри компонента. | Имя выбранного источника данных |

## Атрибуты

| Имя атрибута | Описание                                                                                                               | Значение                     |
|--------------|------------------------------------------------------------------------------------------------------------------------|------------------------------|
| value        | Предустановленное значение внутри компонента.                                                                          | Имя источника данных         |
| size         | Задает размер элемента.                                                                                                | 'small' \| 'middle' \| 'big' |
| required     | Флаг, обязательность значения у компонента. Если значения нет и флаг true, то свойство invalid также будет равно true. | Логическое                   |

## События

| Имя события | Описание                                    |
|-------------|---------------------------------------------|
| input       | Событие изменения данных внутри компонента. |

## Примеры

Для использования компонента достаточно только создать элемент, компонент сам добавит существующие источники данных в список элементов для выбора:

<Canvas>
  <datasource-select></datasource-select>
</Canvas>

<Source
language='html'
format={true}
dark
code={`
<datasource-select></datasource-select>
`}
/>

Кнопка “Создать источник” рядом с выпадающим открывает панель для работы с источниками данных в модальном окне.
