# BaseExpander

Раскрывающийся список (экспандер), представленный тегом `base-expander`. Основан на теге `<details>`.

## Свойства

| Название | Описание                                     | Значение                                                                            |
|----------|----------------------------------------------|-------------------------------------------------------------------------------------|
| theme    | Варианты отображения раскрывающегося списка. | Массив с названиями тем: ['with_borderBottom', 'theme_iconLeft', 'theme_rotate90']. |
| open     | Открыт или раскрыт экспандер.                | Логическое                                                                          |

## Атрибуты

| Название | Описание                                     | Значение                                                                            |
|----------|----------------------------------------------|-------------------------------------------------------------------------------------|
| theme    | Варианты отображения раскрывающегося списка. | Название тем через запятую: 'with_borderBottom', 'theme_iconLeft', 'theme_rotate90'.|

## Слоты

| Название   | Описание                                                                                                                             |
|------------|--------------------------------------------------------------------------------------------------------------------------------------|
| (default)  | Слот по умолчанию. Отвечает за скрытое содержимое.                                                                                   |
| summary    | Слот в элементе, на который нужно кликнуть, чтобы отобразилось скрытое содержимое. Сюда можно поместить текст, кнопки, иконки и т.п. |
| icon-arrow | Слот, в который можно поместить иконку вместо стандартной стрелки.
