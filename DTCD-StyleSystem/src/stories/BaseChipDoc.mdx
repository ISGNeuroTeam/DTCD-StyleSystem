# BaseChip

Компонент для удобной демонстрации каких-либо перечислений, к примеру, фильтров. Представлен тэгом `<base-chip>`. При клике по иконке закрытия происходит генерация события remove.

## Атрибуты

| Название | Описание                                 | Значение   |
|----------|------------------------------------------|------------|
| close    | Показывает или скрывает иконку удаления. | Логическое |

## Слоты

| Название  | Описание                                                       |
|-----------|----------------------------------------------------------------|
| (default) | Слот по умолчанию. Отвечает за внутреннее содержимое элемента. |

## События

| Название | Описание                                  |
|----------|-------------------------------------------|
| remove   | Срабатывает при клике по иконке закрытия. |

## Примеры

По умолчанию, компонент содержит текст “Chip”, который может быть заменен с помощью слота:

<base-chip></base-chip>
<base-chip>Custom chip</base-chip>

```
<base-chip></base-chip>
<base-chip>Custom chip</base-chip>
```

При добавлении атрибута close появляется иконка закрытия, которая генерирует событие remove при нажатии на нее:

<base-chip></base-chip>
<base-chip close></base-chip>

```
<base-chip></base-chip>
<base-chip close></base-chip>
```

Для отлавливания события remove необходимо добавить соответствующий слушатель на элемент. В качестве примера приведен код для VueJS:

```
<template>
  <base-chip close="true" @remove="removeHandler"></base-chip>
</template>
```
```
methods: {
  closeHandler(e) {
    console.log('removed =>', e.target);
  },
},
```
